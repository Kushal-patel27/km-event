# K&M Events (Frontend)

A minimal React + Vite + Tailwind frontend for booking event tickets.

Quick start

1. Install dependencies:

```bash
cd d:/Frontend-EZ
npm install
```

2. Run dev server:

```bash
npm run dev
```
